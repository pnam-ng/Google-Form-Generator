Sign-In Troubleshooting Guide

If you're having trouble signing in with your Google Account, follow these simple steps to resolve common issues.

1. "Google hasn't verified this app" Warning

If you see a warning page from Google saying the app hasn't been verified:

  1. Click on "Advanced" at the bottom of the page
  2. Click "Go to FormGeneration (unsafe)" to continue
  3. You'll be taken to a page where you can grant permissions
  4. Click "Allow" to proceed
Note: This warning appears because the app is new. It's safe to proceed if you trust the application.

2. Can't Sign In

If you're unable to sign in, try these steps:

  1. Make sure you're using a supported web browser (Chrome, Firefox, Edge, or Safari)
  2. Clear your browser cache and cookies
  3. Try signing in using an incognito or private window
  4. Make sure JavaScript is enabled in your browser
  5. Check that you're not blocking pop-ups or redirects

3. Access Denied

If you see an "Access Denied" message:

  • Your Google account may need to be added as a test user
  • Contact the administrator to add your email address
  • Make sure you're using the correct Google account

4. General Tips

  • Use a modern browser: Make sure your browser is up to date
  • Check your internet connection: Ensure you have a stable internet connection
  • Try a different browser: Sometimes switching browsers helps
  • Disable browser extensions: Some extensions can interfere with sign-in
  • Wait a few minutes: If you just made changes, wait 2-5 minutes and try again

Still Having Issues?

If you've tried all the steps above and still can't sign in:

  • Contact support for assistance
  • Provide details about what error message you're seeing
  • Let them know which browser you're using
